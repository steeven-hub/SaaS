# Projet de Tests

Ce dossier centralise tous les tests du projet pour faciliter la maintenance et l'intégration continue.

## Structure
- `/tests/backend` : Tests unitaires et d'intégration pour l'API FastAPI.
- `/tests/frontend` : Tests unitaires (Jest/Karma) pour l'application Angular.

## Exécution
- **Backend** : `pytest tests/backend`
- **Frontend** : `npm test` (depuis le dossier `frontend/`)
